Place the provided logo image here as `logo.jpg`.
Recommended filename: images/logo.jpg
Suggested size: 200×200 or larger (scaled via CSS to 44px height).
If you received the logo as an attachment, save it to this path so the site will display it.

Favicon notes:
- You can reuse the same image for a favicon by saving it as `images/logo.jpg` (browsers accept JPEG favicons when explicitly linked).
- If you prefer a classic `.ico` favicon, create `images/favicon.ico` and include it; browsers will prefer `.ico` when present.
- I added `<link rel="icon" href="images/logo.jpg" type="image/jpeg">` and `<link rel="apple-touch-icon" href="images/logo.jpg">` to each page's `<head>`.
- To create a proper multi-resolution `favicon.ico` from the JPG, use an online converter or an image editor that can export `.ico`.
